Hello!

I am a simple showcase of a web interface for Hyperledger Fabric.

It is recommended to first create a user account by navigating to /register.

Then, using the username you have created, you can navigate to {username}/students to get a full list of students in the DB.

Alternatively, you can go to {username}/students/{student_id} to retrieve a particular student from the DB.
